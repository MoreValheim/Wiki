    Smelter¶

    Smelter is a buildable object found in Valheim.

    Copper
    Type Crafting
    Source Workbench
    Durability ?
    Size ~2x2x4
    To unlock - obtain Surtling Core

    It is possible to speed up the smelting process by filling up the smelter before sleeping. During the sleep phase, the smelter can be heard producing refined ore at an accelerated rate.

    Crafting¶

    Materials Needed
    x 20
    x 5

    How to build:¶

    You need to have a workbench nearby in order to construct a smelter.

    1. Equip your hammer
    2. Right-click
    3. Press [Q/E] to switch to the Crafting tab of the building interface
    4. Left-click the smelter on the displayed page
    5. Left-click to place down
